# BePower

We build the infrastructure for electric mobility.

## Developer Tools

- [`@bepower/dev`](https://github.com/BePower/.github) — CLI tool and golden configs for all BePower projects

## Standards

All our projects use a shared, opinionated setup:

| Tool | Purpose |
|------|---------|
| **Biome** | Linting & formatting |
| **Lefthook** | Git hooks |
| **Vitest** | Testing |
| **tsdown** | Building |
| **commitlint** | Conventional commits |
| **npm** | Package manager |
| **Node >= 22** | Runtime |

See our [Contributing Guide](https://github.com/BePower/.github/blob/main/CONTRIBUTING.md) to get started.
